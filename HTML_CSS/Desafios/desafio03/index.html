Projeto FullStack

O que é um desenvolvedor FullStack

Um desenvolvedor fullstack é aquele profissional que domina tanto o front end quando o back end de uma aplicação web. O desenvolvedor é capaz de contruir a interface que o usuário vê e interage (front end), e também a parte lógica e estrutural que roda por trás dos bastidores (back end).

Esse profissional entende como os dados são procecssados, armazenados e exibidos, conseguindo transitar entre diferentes tecnologias e etapas do desenvolvimento. Ele pode criar desde o layout de uma página até a conexão com banco de dados e servidores.

Camada O que faz Tecnologias comuns
Front end Interface visual e interatividade HMLT, CSS, JavaScript
Back end Lógica de negocios, Banco de dados e Segurança Java, SQL

Porque aprender dos dois?

  1. Autonomia: Você consegue criar projetos completos sem depender de outras pessoas.
  2. Empregabilidade: Empresa valorizam profissionais mais versáteis
  3. Comunicação: Mesmo que você se especialize em uma área, saber como a outra funciona melhora o trabalho em equipe
  4. Criatividade: Você pode tirar ideias do papel com maior liberdade e controle
  5. Visão sistêmica: Entender o fluxo completo; do clique do usuário até a resposta do servidor, te torna um desenvolvedor mais estratégico

Aprender front end te dá do mínio sobre a exepriência do usuário. Aprender back end te dá controle sobre os dados e a lógica. Juntos, eles te transformam em um profissional completo e preparado para qualquer desafio no mundo da tecnologia
