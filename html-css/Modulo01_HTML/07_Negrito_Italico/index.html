Texto grande e pequeno

Observamos que neste texto, existem letras grandes e letras pequenas neste paragrafo. As letras grandes foram geradas apartir da tag <big>, porém está tag está obsoleta e não é recomentado seu uso.

Já as letras pequenas foram criadas com a tag <small> que esta em uso no HTML

Mas porque big se tornou obsoleta e small não?

Na verdade, o HTML manteve o small somente para serem usadas como as famosas letras pequenas de um contrato, sendo esta considerada semântica, enquanto o big não teria utilidade e, caso o desenvolvedor necessite quem uma palavra ou trecho de texto fique com algum tipo de formatação diferenciada, isso deve ser realiado no CSS.
